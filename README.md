Criando um Dashboard gerencial para tomada de decisoes no Power BI
------

Sobre o projeto
----

O Projeto consiste na modidificação de um relatório criativo que já havia sido criado anteriormente, focando na experiência do usuário.

Etapas:
- Modificação do visual do relatório com objetivo de criar um visual mais organizado e com informações essenciais para a analise de tomada de decisão.
- Criação da terceira pagina de relatório.
- Criação de menus de navegação em cada página.
